Ce programme est distribue sous license GPL v3
il est en cours de developpement sous LAZARUS et fourni sans aucune garantie
Attention ce programme ne fonctionne pour l'instant qu'avec le module USB velleman k8062

L'Ide LAZARUS est un projet libre et gratuit telechargeable sur sourceforge.

Ce programme fonctionne avec la gestion des ipc (Shared memory) c'est pourquoi il est compose de 2 programmes dmxd et Dmxcontrol
Vous devez dabord compiler le programme Dmx_2.c en suivant les instructions contenues dans le fichier.
puis lancer le programme et brancher le module k8062 , la led 'DMX signal' du boitier k8062 doit alors clignoter.

Les executables fournis sont valides sur une machine linux x86_64 (fedora 12 noyau 2.6.32 .
Si probleme ,recompilez le programme a l'aide de LAZARUS

!! Le programme dmxcontrol ne demarrera pas si le programme dmxd n'est pas lance. !!

Pour tout autre plateforme ,chargez le projet dmxcontrol dans Lazarus puis compilez le pour votre plateforme.

dmxd peut etre lance manuellement mais pour un lancement automatique de dmxd, le fichier 10-k8062.rules est founi et doit 
etre copie dans /etc/udev/rules.d/ (pour une fedora)

Un fichier exemple 'univers.dmx' est livré avec .

Bonne découverte !!!
